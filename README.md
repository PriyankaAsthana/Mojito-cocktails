## 🧾 Features

- 🧊 Ice-cube drop animation on load
- 🍃 Floating leaves with soft motion
- ✨ Fade-in text reveals and cocktail cards animation
- 📱 Fully responsive layout
- 🎨 Black-mint velvet color palette
- 🖋️ Custom fonts: `Modern Negra`, `DM Serif Text`, `Mona Sans`
- 🎩 Classy motion with GSAP and Framer Motion
- 🧪 Component-based architecture for scalable styling
- 🧠 Lazy loading + route-based code splitting (planned)
- 🖼️ Optimized assets & image responsiveness
